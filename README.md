🌦 Weather Analytics Dashboard (Power BI)
📌 Overview

This project is an interactive Weather Analytics Dashboard built using Power BI. It visualizes real-time and forecast weather data using a Weather API, providing insights into temperature, air quality, and environmental conditions.
❗ Problem Statement

Weather data is often scattered, complex, and difficult to interpret for everyday users. Most platforms provide raw numerical data without meaningful insights, making it hard to understand real-world impact such as air quality risks or daily planning decisions.

There is a need for an interactive system that can:

Convert raw weather and air quality data into clear, visual insights
Provide real-time updates for better decision-making
Offer health-related suggestions based on pollution levels
Present all information in a simple and user-friendly dashboard
🚀 Features
🌡 Real-time temperature & “Feels Like” insights
📊 Forecast analysis (Min/Max trends)
🌫 Air Quality Index (AQI) classification (SO₂, PM10, etc.)
💡 Smart suggestions based on pollution levels
📍 Location-based weather tracking
🎨 Interactive and modern dashboard UI
🛠️ Tech Stack
Power BI – Data Visualization
DAX – Calculated Measures & Logic
Power Query – Data Transformation (ETL)
Weather API – Real-time data source
📊 Data Source
Weather data is fetched using Weather API
Includes:
Current weather
Forecast data
Air quality parameters (SO₂, CO, PM2.5, PM10, etc.)
